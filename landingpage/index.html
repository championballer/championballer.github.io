Welcome to installer.to

A community managed list of installer scripts for popular tools for Linux based OS.

About installer.to

This is a project under the leopardslab organisation making installation of popular tools simple.

Services

Popular tools can be installed on the machine using simple scripts like

  • curl https://installer.to/git | bash
  • curl https://installer.to/docker | bash

Contribute

The project is looking for contributers who have a thing for making tools that help make lives of the community easy. The project repository is hosted on github. For any issues or queries, you can reach out on the gitter channel of scorelab.
